어나더 지오엑스 — 사이트 링크·GitHub 반영 스크립트
================================================

1) site.config.json 설정 (프로젝트 루트)
   - githubPagesBaseUrl : GitHub Pages 공개 주소 (예: https://아이디.github.io/저장소명)
   - githubRepoUrl      : 저장소 URL (선택, HTML에 __SITE_GITHUB_REPO__ 가 있을 때만)
   - gitRoot            : .git이 있는 폴더가 이 프로젝트와 다르면 상대경로 (예: world-climate-study)
   참고: site.config.example.json

2) HTML에 플레이스홀더
   - canonical 등: href="__SITE_GITHUB_PAGES__"
   - sync-site-links.mjs 가 site.config.json 값으로 치환합니다.

3) 실행 방법 (프로젝트 루트에서)
   npm run site:sync     → 링크만 갱신
   npm run site:push     → git add / commit / push 만
   npm run site:publish  → sync 후 push (일반적으로 이것만 쓰면 됨)

   PowerShell:  .\scripts\publish-site.ps1
               .\scripts\publish-site.ps1 -Message "수정 내용"

   CMD:        scripts\publish-site.cmd
               scripts\publish-site.cmd 수정 내용

4) 필요 조건
   - Node.js 설치, git 설치, 원격(origin) 및 인증 설정 완료
   - GitHub Pages는 보통 main 푸시 후 자동 빌드됩니다 (저장소 Settings에서 Pages 소스 확인).
